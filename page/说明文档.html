🧠 AIGC情绪愈疗平台

基于Vue 3的完整心理疗愈解决方案

📋 项目概述

本项目是一个基于人工智能生成内容（AIGC）技术的情绪愈疗辅助平台，通过集成情绪识别、心理风险识别、历史对话分析、情感支持对话和智能商品推荐等功能，为用户提供实时、个性化的情感支持和心理健康服务。

🚀 页面导航

🏠 主应用页面

完整的Vue 3应用，包含所有核心功能

📄 静态页面

原生HTML/CSS/JS实现的静态版本

👤 用户中心

用户信息管理、密码修改、购物车

✨ 核心功能特性

🎯 多模态情绪识别

实时识别用户情绪状态（喜悦、悲伤、愤怒、恐惧等），并以可视化方式展示情绪分析结果。

💬 共情式AI对话

4种不同性格的AI医生选择，提供个性化的情感支持和心理疏导服务。

⚠️ 心理风险预警

智能检测高风险词汇，自动触发预警机制并提供危机干预资源。

📊 心理健康报告

生成多维度心理评估报告，包括焦虑、抑郁、睡眠、压力等评估。

📈 数据可视化

使用ECharts展示心理健康趋势图表，直观了解心理状态变化。

🛒 智能商品推荐

根据用户情绪状态和心理健康需求，推荐个性化的疗愈商品。

👤 用户中心

完整的用户管理系统，包括个人信息编辑、密码修改、购物车管理。

📱 响应式设计

完美适配PC和移动端，提供一致的用户体验。

🛠️ 技术栈

Vue 3.4.21 Vue Router 4.3.0 ECharts 5.5.0 HTML5 CSS3 JavaScript ES6+ LocalStorage 响应式设计

🎮 快速开始

💡 使用提示

  • 点击页面右上角的👤用户头像，可进入用户中心
  • 在用户中心可以管理个人信息、修改密码、查看购物车
  • 登录用户名：admin，密码：123456
  • 支持匿名体验模式，无需注册即可使用
  • 所有数据保存在浏览器本地，刷新页面不会丢失
  • 购物车商品数量会实时同步到用户中心

🔄 页面间跳转

从主页面到用户中心：点击页面右上角的用户头像即可跳转到用户中心。
